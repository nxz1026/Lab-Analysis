# 高级分析功能说明

## 📋 概述

`data_analyzer.py` 新增了3个高级统计分析功能，提供更深入的医学检验数据洞察。

---

## ✨ 新增功能

### **1. 移动平均分析（Moving Average）** 📈

**功能描述：**
- 计算滚动平均值和标准差
- 平滑短期波动，识别真实趋势
- 可自定义窗口大小（默认2次就诊）

**临床意义：**
- 消除单次测量的随机误差
- 更清晰地展示指标的长期变化趋势
- 帮助判断治疗效果是否持续

**输出示例：**
```json
{
  "hs-CRP": {
    "moving_avg": [2.78, 2.095, 1.13, 0.685],
    "rolling_std": [0.0, 0.969, 0.396, 0.233],
    "trend": "下降",
    "window": 2
  }
}
```

**实现函数：** `moving_average_analysis(df, window=2)`

---

### **2. 变异系数（CV）稳定性分析** 📊

**功能描述：**
- 计算每个指标的变异系数（CV = 标准差/均值）
- 评估指标的生物学稳定性
- 识别高波动风险指标

**稳定性分类：**
- ✅ **稳定**：CV < 0.1（低风险）
- 🟡 **中等变异**：0.1 ≤ CV < 0.2（中风险）
- ⚠️ **高变异**：CV ≥ 0.2（高风险，需关注）

**临床意义：**
- 评估患者生理状态的稳定性
- 识别需要密切监测的指标
- 区分生理性波动 vs 病理性变化

**输出示例：**
```json
{
  "hs-CRP": {
    "cv": 0.7171,
    "mean": 1.390,
    "std": 0.997,
    "stability": "高变异",
    "risk_level": "高",
    "n_points": 4,
    "rolling_cv": [null, null, 0.6823, 0.5941]
  }
}
```

**实现函数：** `cv_stability_analysis(df)`

---

### **3. Z-score异常检测** 🚨

**功能描述：**
- 基于标准化分数（Z-score）识别异常值
- 分级检测：轻度异常（|Z|>2）和严重异常（|Z|>3）
- 量化偏离正常范围的程度

**异常分级：**
- ⚠️ **轻度异常**：|Z| > 2.0（统计学显著偏离）
- 🚨 **严重异常**：|Z| > 3.0（极端异常值）

**临床意义：**
- 自动识别异常就诊记录
- 量化指标偏离程度
- 辅助发现潜在的病理状态变化

**输出示例：**
```json
{
  "CRP": {
    "z_scores": [1.055, 0.576, -0.48, -1.151],
    "outliers_mild": {
      "count": 0,
      "dates": [],
      "values": []
    },
    "outliers_severe": {
      "count": 0,
      "dates": [],
      "values": []
    },
    "max_deviation": {
      "date": "2026-03-24",
      "value": 10.0,
      "z_score": 1.055
    },
    "threshold": 2.0
  }
}
```

**实现函数：** `zscore_outlier_detection(df, threshold=2.0)`

## 🔧 使用方法

### **方式一：完整Pipeline（自动执行）**

```bash
python -m lab_analysis.pipeline --patient-id YOUR_PATIENT_ID
```

所有新分析会自动包含在：
- `analysis_results.json`（JSON格式）
- `analysis_results_report.md`（Markdown报告）

### **方式二：单独调用 data_analyzer**

```bash
python -m lab_analysis.data_analyzer --patient-id YOUR_PATIENT_ID
```

### **方式三：代码中调用**

```python
from lab_analysis.data_analyzer import (
    moving_average_analysis,
    cv_stability_analysis,
    zscore_outlier_detection
)

# 移动平均分析
ma_results = moving_average_analysis(df, window=2)

# CV稳定性分析
cv_results = cv_stability_analysis(df)

# Z-score异常检测
zscore_results = zscore_outlier_detection(df, threshold=2.0)
```

## 🎯 参数调优建议

### **移动平均窗口大小**

| 窗口大小 | 适用场景 | 优点 | 缺点 |
|---------|---------|------|------|
| 2 | 短期趋势 | 响应快 | 噪声较多 |
| 3 | 中期趋势 | 平衡性好 | - |
| 4+ | 长期趋势 | 平滑度高 | 延迟大 |

**推荐：** 对于4次就诊数据，使用`window=2`；对于10+次就诊，可使用`window=3-4`。

### **Z-score阈值**

| 阈值 | 敏感度 | 适用场景 |
|------|--------|---------|
| 1.5 | 高 | 早期预警 |
| 2.0 | 中 | **默认推荐** |
| 3.0 | 低 | 仅关注极端异常 |

**推荐：** 保持默认`threshold=2.0`，平衡敏感度和特异性。

### **数学公式**

**移动平均：**
```
MA_t = (x_t + x_{t-1} + ... + x_{t-w+1}) / w
```

**变异系数：**
```
CV = σ / μ
```

**Z-score：**
```
Z = (x - μ) / σ
```
