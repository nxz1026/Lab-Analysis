# 高级分析可视化功能

## 📊 概述

为3个高级统计分析功能添加了专业的可视化图表，使数据分析结果更直观易懂。

---

## 🎨 新增图表

### **图5：移动平均趋势图** (`fig_05_moving_average.png`)

**图表类型：** 多子图折线图（2×2布局）

**展示内容：**
- 🔵 **蓝色实线**：原始测量值
- 🔴 **红色虚线**：移动平均值（窗口=2）
- 🔴 **浅红阴影**：±1标准差区域

**临床价值：**
- 直观对比原始数据和平滑趋势
- 清晰展示指标的长期变化方向
- 标准差区域反映波动程度

**示例指标：** hs-CRP, CRP, WBC, NEUT#

---

### **图6：CV稳定性热力图** (`fig_06_cv_stability.png`)

**图表类型：** 水平条形热力图

**展示内容：**
- 🟢 **绿色条**：稳定指标（CV < 0.1）
- 🟡 **黄色条**：中等变异（0.1 ≤ CV < 0.2）
- 🔴 **红色条**：高变异指标（CV ≥ 0.2）
- 每个条形上标注具体CV值和风险等级

**临床价值：**
- 一眼识别需要重点关注的指标
- 量化评估各指标的稳定性
- 辅助制定个性化监测计划

**排序方式：** 按指标名称字母排序

---

### **图7：Z-score分布图** (`fig_07_zscore_distribution.png`)

**图表类型：** 箱线图（Box Plot）

**展示内容：**
- 📦 **箱体**：Z-score的四分位距（IQR）
- ➖ **中位线**：Z-score中位数
- ⚪ **离群点**：异常值标记
- 🔴 **虚线**：|Z|=2 阈值（轻度异常）
- 🔴 **点线**：|Z|=3 阈值（严重异常）
- 🟢 **绿色背景**：正常范围（|Z|<2）

**颜色编码：**
- 🔵 **蓝色箱体**：无异常值（所有|Z|<2）
- 🔴 **红色箱体**：存在异常值（至少一个|Z|≥2）

**临床价值：**
- 统计学方法识别异常就诊记录
- 量化偏离正常范围的程度
- 发现潜在的病理状态转变

**示例指标：** hs-CRP, CRP, WBC, NEUT#, MONO%, RDW-SD

## 💡 图表解读指南

### **移动平均趋势图解读**

**场景1：炎症指标持续下降**
```
hs-CRP: ↓ 下降趋势
- 原始值波动较大（2.78 → 1.41 → 0.85 → 0.52）
- 移动平均平滑后显示清晰下降趋势
- 标准差区域逐渐收窄 → 稳定性提高

临床意义：治疗有效，炎症得到控制
```

**场景2：血常规指标平稳**
```
WBC: → 平稳趋势
- 原始值在小范围内波动（6.7 → 7.2 → 6.9 → 7.5）
- 移动平均接近水平线
- 标准差区域窄 → 波动小

临床意义：骨髓造血功能稳定
```

---

### **CV稳定性热力图解读**

**高风险指标示例：**
```
hs-CRP: CV=0.7171 [高变异] ⚠️
- 炎症指标波动大是正常的（病情变化）
- 但需密切监测，防止急性发作

CRP: CV=0.4668 [高变异] ⚠️
- 同样反映炎症状态不稳定
- 建议缩短随访间隔
```

**稳定指标示例：**
```
WBC: CV=0.0495 [稳定] ✅
NEUT#: CV=0.0833 [稳定] ✅
- 血常规指标稳定
- 可按常规随访计划
```

---

### **Z-score分布图解读**

**正常情况：**
```
所有指标箱体在 ±2 范围内
- 无离群点
- 箱体颜色为蓝色

临床意义：患者生理状态稳定
```

**异常情况：**
```
某指标出现离群点（|Z|>2）
- 箱体颜色变为红色
- 离群点对应特定就诊日期

临床意义：该次就诊可能存在：
1. 急性病情变化
2. 测量误差
3. 治疗方案调整期
→ 需要结合临床进一步分析
```

## 🎯 临床应用案例

### **案例1：慢性胰腺炎治疗效果评估**

**背景：** 患者接受4次就诊，监测炎症指标

**图表分析：**
1. **移动平均图**：hs-CRP和CRP均显示下降趋势
2. **CV热力图**：炎症指标为高变异（预期），血常规稳定
3. **Z-score图**：首次就诊Z=1.39（接近异常），后续均在正常范围

**临床决策：**
- ✅ 治疗有效（炎症指标下降）
- ⚠️ 继续监测（高变异性提示未完全稳定）
- 📅 2周后复查

---

### **案例2：早期预警系统**

**图表表现：**

- 移动平均图显示WBC上升趋势
- CV分析可能显示稳定性下降

**临床行动：**
- 🚨 立即评估感染风险
- 🔬 追加细菌培养检查
- 💊 考虑预防性抗生素

## 📈 技术细节

### **图表规格**

| 图表 | 尺寸 | DPI | 格式 |
|------|------|-----|------|
| 移动平均图 | 12×8英寸 | 150 | PNG |
| CV热力图 | 10×(n×0.6+2)英寸 | 150 | PNG |
| Z-score图 | (n×1.5+2)×6英寸 | 150 | PNG |

*注：n = 指标数量*

### **颜色方案**

- **主色调**：蓝色系 (#3498db) - 原始数据/正常状态
- **强调色**：红色系 (#e74c3c) - 趋势线/异常状态
- **辅助色**：绿色系 (#27ae60) - 正常范围/稳定状态
- **警告色**：黄色系 (#f39c12) - 中等风险

### **字体设置**

- 中文字体：Microsoft YaHei / SimHei
- 英文字体：DejaVu Sans
- 标题字号：13-14pt（加粗）
- 标签字号：9-11pt

## 🔧 自定义选项

### **修改移动平均窗口**

在 `data_analyzer.py` 中修改：
```python
ma_results = moving_average_analysis(df, window=2)  # 改为3或4
```

### **调整Z-score阈值**

```python
zscore_results = zscore_outlier_detection(df, threshold=2.0)  # 改为1.5或2.5
```

### **选择绘图指标**

在绘图函数中修改 `key_metrics` 列表：
```python
key_metrics = ["hs-CRP", "CRP", "WBC", "NEUT#", "PLT"]  # 添加更多指标
```
